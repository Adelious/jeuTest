# TestJeu
 This is a test of a game that I made in Unity.

## Controls

### Movements

* **Z** : Go Forward
* **S** : Go Backward
* **Q** : Strafe left
* **D** : Strafe Right

### Other

* **Escape** : Toggle pause menu

### Developer

* **F1** : Debug console

## Additions

* A player which move with **root motion** and that legs are enhanced with **foot ik**
* A Networking system
